# RSA-Network-Security-Project

This project demonstrates the RSA algorithm in C for a network security course project. It generates RSA keys from two prime numbers, reads plaintext from a file, encrypts each character using the private key, and then decrypts the ciphertext using the public key.

## Files

- `rsa.c` - main C source file
- `plaintext.txt` - input message to encrypt
- `ciphertext.txt` - encrypted integer output generated by the program
- `decoded.txt` - decrypted output generated by the program

## RSA Steps Implemented

1. Choose two prime numbers `p` and `q`
2. Compute `n = p * q`
3. Compute `phi = (p - 1) * (q - 1)`
4. Choose `e` such that `gcd(e, phi) = 1`
5. Compute `d` as the modular inverse of `e mod phi`
6. Encrypt with the private key `(d, n)`
7. Decrypt with the public key `(e, n)`

## Build and Run

### Using GCC

```bash
gcc rsa.c -o rsa.exe
./rsa.exe
```

### Using MSVC Developer Command Prompt

```bat
cl rsa.c
rsa.exe
```

## Input Methods

When you run the program, you now have two choices:

1. **Use `plaintext.txt`** as before
2. **Type a message from the keyboard** and the program will save it into `plaintext.txt` automatically

## Notes

- The program uses modular exponentiation for efficient RSA operations.
- The modulus `n` is larger than 16 bits, satisfying the project requirement.
- This implementation is for educational demonstration, not real-world secure deployment.
